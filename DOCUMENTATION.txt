
Документацию можно найти по адресу http://zoomscroll.ru


1. Подключение:

	<script type="text/javascript" src="jquery-1.6.4.min.js"></script>  <!-- библиотека jQuery -->
	<script type="text/javascript" src="jquery.mousewheel.min.js"></script>  <!-- добавляет поддержку колеса мыши -->
	<script type="text/javascript" src="jquery-ui-1.8.16.custom.min.js"></script>  <!-- jQuery UI: draggable -->
	
	<link rel="stylesheet" type="text/css" href="jquery.zoomscroll-2.0.1.min.css" />  <!-- стили для zoomscroll -->
	<script type="text/javascript" src="jquery.zoomscroll-2.0.1.min.js"></script>  <!-- сама библиотека zoomscroll -->

	
	
	
2. Как пользоваться:

	$( function (){
		
		// подключаем zoomscroll
		$("a.picture").zoomscroll();
		
	})
	
	
	
	
3. Управление:

	Колесо мыши вверх — приближает изображение.
	Колесо мыши вниз — отдаляет изображение.
	Двойной щелчок мышью — возвращает изображение в исходное положение.
	Захват мышью изображения и перемещение — двигает картинку.
